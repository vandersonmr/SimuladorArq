Welcome to the SimulatorMIPS page.

This is architecture MIPS simulator developed how a academic work of the Computer Science course at UEM. The specification of the architecture can be found with the source code. In this program you can enter with a assembly code (In the specification) and using the mont pass it to hexdecimal and run in the simulator. You also can run step by step and analyze the values in the registers, memory and/or in the pipeline stages. This simulator is on freeBSD license.
